# Portfolio Assignment - Raghada Alqarawi-202257820


A responsive and interactive personal portfolio website developed as part of Assignment 2. This version enhances the first assignment by adding dynamic behavior, improved user experience, and better code structure.

---

## Features

### Responsive Design
- Fully responsive across Mobile, Tablet, and Desktop devices.
- Uses CSS Grid and Flexbox for adaptive layouts.

### Advanced Interactivity
- Dark/Light Theme Toggle with localStorage persistence.
- Dynamic Greeting Banner based on time of day.
- Smooth Scrolling Navigation between sections.
- Active Navigation Highlight on scroll.
- Scroll Reveal Animations using IntersectionObserver.

### Dynamic Project System
- Filter projects by category (Planning, Navigation, Productivity).
- Real-time Search functionality for project titles.
- Dynamic "No Results" message when no projects match.



---

## Modern Web Techniques Used

- CSS Variables for theme switching
- LocalStorage for state persistence
- IntersectionObserver API for animations
- Event-driven JavaScript (Event Listeners)
- DOM Manipulation for dynamic UI updates
