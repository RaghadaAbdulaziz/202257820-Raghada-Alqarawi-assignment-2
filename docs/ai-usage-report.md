# AI Usage Report

## Tools Used & Use Cases

### Claude
Claude was used as the primary tool for code generation and debugging. It assisted in:
- Writing and improving JavaScript functions for dynamic features such as project filtering, search functionality, and modal interaction.
- Reviewing and optimizing existing code structure to ensure better organization and readability.
- Debugging issues related to event handling and DOM manipulation.

### Google Gemini
Google Gemini was used mainly for documentation support and UI/UX enhancement. It assisted in:
- Structuring the README and technical documentation.
- Suggesting improvements for user experience, such as adding clear interaction guidance and better form structure.
- Providing recommendations for layout consistency and visual clarity.

---

## Benefits & Challenges

### Benefits
Using AI significantly improved development efficiency. It helped transform the portfolio from a static website into a more interactive application by suggesting dynamic features such as filtering, searching, and modal popups. It also improved code readability and reduced debugging time.

### Challenges
One challenge was ensuring that AI-generated solutions remained simple and aligned with assignment requirements. Some suggestions included unnecessary complexity or external libraries. This was resolved by refining prompts and focusing only on standard HTML, CSS, and JavaScript solutions.

---

## Learning Outcomes

Through using AI tools, I learned how to:
- Implement dynamic UI features using JavaScript (filtering, searching, modal systems).
- Improve user experience by adding real-time feedback and interactivity.
- Structure JavaScript code into modular and reusable functions.

---

## Responsible Use & Modifications

Although AI tools assisted in generating and improving code, I did not rely on them blindly. I reviewed, tested, and modified all outputs to ensure correctness and alignment with my assignment. I ensured that all features were implemented using standard web technologies as required by the assignment. I also adjusted the design and content to reflect my own projects and personal style.
